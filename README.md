# 🎬 映画クイズ道場

映画の知識を楽しみながら学べるクイズゲームです。

## 🎮 遊び方

1. ジャンルを選択してプレーヤー名を入力
2. 120秒の制限時間で10問に挑戦
3. **正解** → +10点 ＋ 時間+5秒
4. **不正解** → 減点なし ＋ 時間-5秒
5. 残り時間がそのままボーナス点に加算！

## 📚 ジャンル

| ジャンル | 説明 |
|---------|------|
| 🗾 日本映画 | 邦画の名作・話題作 |
| 🌎 外国映画 | ハリウッド・世界の映画 |
| 📅 制作年代別 | 各時代の映画を出題 |
| 🎥 監督別 | 名監督の作品 |
| 🌟 俳優別 | 名俳優に関するクイズ |
| 💀 上級者向け | マニアックな難問 |

## ✨ 特徴

- 4択クイズ＋○×ミックス（ランダム出題）
- 350問以上収録
- スコアランキング（ローカル保存）
- 間違えた問題の解説表示
- スマートフォン対応（レスポンシブ）
